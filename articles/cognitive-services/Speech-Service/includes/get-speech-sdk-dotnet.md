#### <a name="c-nuget-package"></a>C# NuGet 패키지

다음 `dotnet add` 명령을 사용하여 **.NET Core CLI** 에서 .NET Speech SDK를 설치할 수 있습니다.

```dotnetcli
dotnet add package Microsoft.CognitiveServices.Speech
```

다음 `Install-Package` 명령을 사용하여 **패키지 관리자** 에서 .NET Speech SDK를 설치할 수 있습니다.

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```

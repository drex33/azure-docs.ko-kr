# <a name="make-resources-more-secure-with-azure-advisor"></a>Azure Advisor로 리소스를 보다 안전하게 만들기

Azure Advisor는 모든 Azure 리소스에 대한 권장 사항을 일관되고 통합된 보기로 표시합니다. Microsoft Defender for Cloud와 통합 되어 보안 권장 사항을 가져옵니다. Advisor 대시보드의 **보안** 탭에서 보안 관련 권장 지침을 얻을 수 있습니다.

클라우드 용 Defender는 Azure 리소스의 보안에 대 한 향상 된 가시성과 제어를 통해 위협을 예방, 감지 및 대응 하는 데 도움이 됩니다. 여기서는 Azure 리소스의 보안 상태를 주기적으로 분석합니다. Defender for Cloud가 잠재적인 보안 취약점을 식별 하는 경우 권장 사항을 만듭니다. 권장 사항은 필요한 컨트롤을 구성하는 과정을 안내합니다. 

보안 권장 사항에 대 한 자세한 내용은 [클라우드 용 Microsoft Defender에서 보안 권장 사항 검토](../defender-for-cloud/review-security-recommendations.md)를 참조 하세요.

## <a name="how-to-access-security-recommendations-in-azure-advisor"></a>Azure Advisor에서 보안 권장 사항에 액세스하는 방법

1. [Azure Portal](https://portal.azure.com)에 로그인하고 [Advisor](https://aka.ms/azureadvisordashboard)를 엽니다.

2.  Advisor 대시보드에서 **보안** 탭을 클릭합니다.

# <a name="configure-saas-offer-listing-details"></a>SaaS 제품 목록 세부 정보 구성

**제품 목록** 페이지에서 SaaS(Software as a Service)에 대해 입력한 정보는 Microsoft 상업용 Marketplace 온라인 스토어에 표시됩니다. 여기에는 제품, 스크린샷 및 마케팅 자산에 대한 설명이 포함됩니다.

> [!NOTE]
> 제품 설명이 “이 애플리케이션은 [영어가 아닌 언어]에서만 사용할 수 있습니다”라는 문구로 시작하는 경우 제품 목록 내용을 영어로 작성하지 않아도 됩니다. 제안 목록 콘텐츠에 사용된 언어가 아닌 다른 언어로 된 제안 콘텐츠에 대한 ‘유용한 링크 URL’을 제공할 수도 있습니다.

## <a name="marketplace-details"></a>마켓플레이스 세부 정보

**제품 목록** 페이지의 **Marketplace 세부 정보** 에서 다음 단계를 완료합니다. 해당 필수 세부 정보에 대한 자세한 내용은 [제품 목록 세부 정보](plan-azure-application-offer.md#offer-listing-details)를 참조하세요.

1. **이름** 상자는 이전에 **새 제품** 대화 상자에서 입력한 이름으로 미리 채워집니다. 언제든지 이름을 변경할 수 있습니다.
1. **검색 결과 요약** 상자에서 텍스트를 입력합니다(최대 50자). 해당 요약문은 마켓플레이스에서 검색 결과를 나열하는 데 사용됩니다.
1. **설명** 상자에 제품 설명을 입력합니다. 이 텍스트 상자에는 보다 설득력 있는 설명을 도와주는 다양한 텍스트 편집기 컨트롤이 들어 있습니다. HTML 태그를 사용하여 설명의 서식을 지정할 수도 있습니다. 이 상자에는 HTML 태그와 공백을 포함하여 최대 3,000자의 텍스트를 입력할 수 있습니다. HTML 서식 지정에 대한 자세한 내용은 [상업용 마켓플레이스 제품 설명에서 지원되는 HTML 태그](supported-html-tags.md)를 참조하세요.
1. **시작 지침** 상자에 고객이 SaaS 제품에 연결하는 데 도움이 되는 지침을 입력합니다. 보다 자세한 온라인 설명서에 최대 3000자의 텍스트와 링크를 추가할 수 있습니다.
1. (선택 사항) **검색 키워드** 상자에 고객이 상업용 Marketplace에서 제품을 찾는 데 사용할 수 있는 최대 3개의 검색 키워드를 입력합니다. 제품 **이름** 및 **설명** 은 포함할 필요가 없습니다. 해당 텍스트는 검색에 자동으로 포함됩니다.
1. **개인정보처리방침 링크** 상자에 조직의 개인 정보 처리 방침에 대한 링크(https로 시작)를 입력합니다. 게시자는 앱이 개인정보법 및 규정을 준수하는지 확인하고 유효한 개인정보처리방침을 제공할 의무가 있습니다.

## <a name="add-supplemental-links-optional"></a>보충 링크 추가(선택 사항)

보충 온라인 설명서에 대한 링크를 추가하려면 다음 단계를 완료합니다.

1. 앱 또는 관련 서비스에 대한 선택적 보충 온라인 설명서를 추가하려면 **유용한 링크** 에서 **링크 추가** 를 선택합니다.
1. 표시되는 필드에 제목(최대 255자)을 입력하고 온라인 설명서에 대한 링크(https로 시작)를 입력합니다.
1. 다른 링크를 입력하려면 1~2단계를 반복합니다.

## <a name="enter-your-contact-information"></a>연락처 정보를 입력합니다.

**연락처 정보** 에서 다음 연락처에 대한 정보를 제공합니다.

- **지원 연락처**(필수) – 일반적인 지원 질문을 위한 연락처입니다.
- **엔지니어링 연락처**(필수) - 기술 관련 질문을 위한 연락처입니다. 인증 문제를 비롯하여 제품에 문제가 있는 경우 이 정보를 사용해 사용자에게 연락합니다.
- **CSP 프로그램 연락처**(선택 사항) – 지원 및 비즈니스 문제를 위한 연락처입니다. 이 정보는 CSP 파트너에만 표시됩니다.

연락처마다 이름, 전화번호 및 메일 주소를 입력합니다(공개적으로 표시되지 않음). **지원 URL** 은 **지원 연락처** 에 필요합니다(공개적으로 표시됨).

1. **지원 연락처** 상자에 이름, 메일 주소, 전화번호 및 지원 페이지 URL을 입력합니다.
1. **엔지니어링 연락처** 상자에 이름, 메일 주소 및 전화번호를 입력합니다.
1. (선택 사항) **CSP 프로그램 연락처** 상자에 이름, 메일 주소 및 전화번호를 입력합니다.
1. [CSP(클라우드 솔루션 공급자) 프로그램](cloud-solution-providers.md)으로 제품을 확장하려면 **CSP 프로그램 마케팅 자료** 상자에 마케팅 자료에 대한 링크를 입력합니다.

   > [!NOTE]
   > CSP 프로그램은 CSP 파트너가 제품을 번들링, 마케팅 및 재판매할 수 있도록 하여 제품을 더 광범위한 범위의 고객으로 확장합니다. 이러한 재판매인은 제품 마케팅 자료에 액세스해야 합니다. 자세한 내용은 [Microsoft를 사용하여 출시하기](https://partner.microsoft.com/reach-customers/gtm)를 참조하세요.

## <a name="add-marketing-documents"></a>마케팅 문서 추가

제품에 1~3개의 마케팅 문서를 추가해야 하며, 해당 문서는 .PDF 형식이어야 합니다.

1. **지원 문서** 에서 PDF 파일을 회색 상자에 끌어다 놓습니다.
1. 표시되는 **이름** 상자에 이 문서의 이름을 입력합니다. 이 이름은 제품의 세부 정보 페이지에서 고객에게도 표시됩니다.
1. 해당 단계를 반복하여 추가 지원 문서를 추가합니다.

## <a name="add-marketplace-media"></a>Marketplace 미디어 추가

### <a name="add-logos"></a>로고 추가

**로고** 에서 216 x 216, 350 x 350픽셀 사이의 PNG 형식으로 **큰** 로고를 업로드합니다. 파트너 센터는 **작은**(48 x 48) 및 **보통**(90 x 90) 로고를 자동으로 만듭니다. 이 로고는 나중에 원하는 경우 바꿀 수 있습니다.

세 가지 로고 크기는 온라인 스토어의 여러 위치에서 사용됩니다.

- **보통** 로고는 Microsoft Azure에서 새 리소스를 만들 때 표시됩니다.
- **큰** 로고는 Azure Marketplace 및 Microsoft AppSource의 제품 목록 페이지에 표시됩니다.
- **작은** 로고는 Azure Marketplace 검색 결과 및 Microsoft AppSource 기본 페이지와 검색 결과 페이지에 표시됩니다.

### <a name="add-screenshots"></a>스크린샷 추가

제품을 설명하는 최소 1개, 최대 5개의 스크린샷을 추가해야 합니다. 이미지들은 각각 1280 x 720픽셀로 .PNG 형식이어야 합니다.

1. **스크린샷** 에서 .PNG 파일을 **스크린샷** 상자에 끌어 놓습니다.
2. **이미지 캡션 추가** 옆의 편집 아이콘을 클릭합니다.
3. 표시되는 대화 상자에서 캡션을 입력합니다.
4. 1~3단계를 반복하여 스크린샷을 추가합니다.

### <a name="add-videos-optional"></a>비디오 추가(선택 사항)

제품을 설명하는 YouTube 또는 Vimeo 비디오에 대한 링크를 추가할 수 있습니다. 이러한 비디오는 제품과 함께 고객에게 표시됩니다. 비디오의 썸네일 이미지를 1280 x 720픽셀 크기, .PNG 형식으로 입력해야 합니다. 제품당 비디오를 최대 4개까지 추가할 수 있습니다.

1. **비디오** 에서 **비디오 추가** 링크를 선택합니다.
2. 표시되는 상자에 비디오의 이름과 링크를 입력합니다.
3. 회색 **썸네일** 상자로 .PNG 파일(1280 x 720픽셀)을 끌어서 놓습니다.
4. 다른 비디오를 추가하려면 1~3단계를 반복합니다.

> [!TIP]
> 파일을 업로드하는 동안 문제가 발생한 경우 로컬 네트워크가 파트너 센터에서 사용하는 [ **https://upload.xboxlive.com** ](https://upload.xboxlive.com/) 서비스를 차단하지 않는지 확인하세요.

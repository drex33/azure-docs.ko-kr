# <a name="set-the-preview-audience-for-an-iot-edge-module-offer"></a>IoT Edge 모듈 제품의 미리 보기 대상 설정

**미리 보기 대상** 을 선택하여 제품을 광범위한 Marketplace 대상에 게시하기 전에 유효성을 검사하기 위해 제한된 미리 보기 대상을 선택할 수 있습니다.

> [!IMPORTANT]
> 미리 보기에서 제품을 본 후 **라이브로 전환** 을 선택하여 제품을 공개 게시해야 합니다.

## <a name="select-preview-audience-guids"></a>미리 보기 대상 GUID 선택

하나 이상의 Azure 구독 ID GUID와 선택적 설명과 함께 미리 보기 대상을 지정합니다. Azure Portal의 구독 페이지에서 Azure 구독 ID를 찾을 수 있습니다. 해당 필드는 고객에게 표시되지 않습니다.

개별적으로 최대 10개의 GUID를 추가하거나 CSV 파일을 업로드하여 최대 100개의 GUID를 추가합니다. 이러한 구독 ID를 추가하여 제품이 라이브로 게시되기 전에 제품을 미리 볼 수 있는 사용자를 정의합니다. 제품이 이미 라이브 상태인 경우 제품의 변경 내용 또는 업데이트를 테스트하는 미리 보기 대상 그룹을 정의할 수 있습니다.

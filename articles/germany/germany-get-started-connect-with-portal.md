# <a name="connect-to-azure-germany-by-using-the-azure-portal"></a>Azure Portal을 사용하여 Azure 독일에 연결

[!INCLUDE [closureinfo](../../includes/germany-closure-info.md)]

[Azure Portal](https://portal.microsoftazure.de/)은 대부분의 사람들이 Azure 독일에 연결에 사용하는 주요 방법입니다. 

> [!NOTE]
> Azure 독일에서는 새 포털(Azure Resource Manager 배포 모델을 사용하는 포털)만 사용할 수 있습니다.
>

[계정 홈 페이지](https://account.windowsazure.de)로 이동하여 계정에 대한 구독을 만들 수 있습니다.

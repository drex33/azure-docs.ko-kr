## <a name="next-steps"></a>다음 단계

이 빠른 시작에서는 *클라이언트 URL 생성기* 를 사용하여 일시적으로 사용 가능한 클라이언트 URL을 생성하여 서비스에 연결하고 Web PubSub 서비스에 대한 몇 가지 기본 아이디어를 제공합니다.

실제 애플리케이션에서는 다양한 언어의 SDK를 사용하여 고유한 애플리케이션을 빌드할 수 있습니다. 또한 서버리스 애플리케이션을 쉽게 빌드할 수 있도록 함수 확장도 제공합니다.

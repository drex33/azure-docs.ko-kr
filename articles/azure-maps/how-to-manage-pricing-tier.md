# <a name="manage-the-pricing-tier-of-your-azure-maps-account"></a>Azure Maps 계정의 가격 책정 계층 관리

Azure Portal을 통해 Azure Maps 계정의 가격 책정 계층을 관리할 수 있습니다. [계정](https://azure.microsoft.com/free/?WT.mc_id=A261C142F)을 만들면 계정의 가격 책정 계층을 보거나 변경할 수도 있습니다.

[Azure Maps에서 적절한 가격 책정 계층 선택](./choose-pricing-tier.md)에 관해 알아봅니다.

>[!NOTE]
>Gen 2 Azure Maps Creator 고객의 경우 Gen 1 가격 책정 계층으로 전환할 수 없습니다.  Gen 1 Azure Maps Creator는 2021년 8월 6일에 더 이상 사용되지 않습니다.

## <a name="view-your-pricing-tier"></a>가격 책정 계층 보기

선택한 가격 책정 계층을 보려면 설정 메뉴에서 **가격 책정 계층** 옵션으로 이동합니다.

:::image type="content" source="./media/how-to-manage-pricing-tier/view-pricing-tier.png" border="true" alt-text="선택한 가격 책정 계층 보기":::

## <a name="change-a-pricing-tier"></a>가격 책정 계층 변경

Azure Maps 계정을 만든 후 Azure Maps 계정에 대한 가격 책정 계층을 업그레이드하거나 다운그레이드할 수 있습니다. 업그레이드하거나 다운그레이드하려면 설정 메뉴에서 **가격 책정 계층** 옵션으로 이동합니다. 드롭다운 목록에서 가격 책정 계층을 선택합니다.  참고 – 현재 가격 책정 계층이 기본 선택입니다.  **저장** 단추를 선택하여 선택한 가격 책정 계층 옵션을 저장합니다.

> [!NOTE]
> Azure Maps 계정에 대한 가격 책정 계층을 업그레이드 또는 다운그레이드하는 경우 새 구독 키 또는 클라이언트 ID(Azure AD 인증용)를 생성할 필요가 없습니다.


:::image type="content" source="./media/how-to-manage-pricing-tier/change-pricing-tier.png" border="true" alt-text="가격 책정 계층 변경":::

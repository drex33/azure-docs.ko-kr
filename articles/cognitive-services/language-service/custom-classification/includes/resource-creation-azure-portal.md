### <a name="create-a-new-resource-from-the-azure-portal"></a>Azure Portal에서 새 리소스를 만듭니다.

[Azure Portal](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesTextAnalytics)로 이동하여 새 Azure 언어 리소스를 만듭니다. 추가 기능을 선택하라는 메시지가 표시되면 **이 단계 건너뛰기** 를 선택합니다. 리소스를 만들 때 다음 매개 변수가 있는지 확인합니다.  

|요구 사항  |필수 값  |
|---------|---------|
|위치 | "West US 2" 또는 "West Europe"         |
|가격 책정 계층     | 표준(**S**) 가격 책정 계층        |

> [!IMPORTANT]
> 사용자 지정 **NER (명명 된 엔터티 인식) & 사용자 지정 분류 (미리 보기)** 섹션에서 기존 저장소 계정을 선택 하거나 새 계정을 만들어야 합니다. 저장소 계정은 사용자 지정 텍스트 분류를 사용 하는 데 필요 합니다. 나중에 저장소 계정을 지정할 수 있지만 지금 더 쉽게 수행할 수 있습니다.

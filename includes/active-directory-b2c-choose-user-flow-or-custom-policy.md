Azure Active Directory B2C는 사용자가 애플리케이션과 상호 작용하는 방법을 정의하는 두 가지 방법, 즉 미리 정의된 [사용자 흐름](../articles/active-directory-b2c/user-flow-overview.md) 또는 완전히 구성 가능한 [사용자 지정 정책](../articles/active-directory-b2c/custom-policy-overview.md)을 통해 제공합니다. 이 문서에서 필요한 단계는 각 방법마다 다릅니다.

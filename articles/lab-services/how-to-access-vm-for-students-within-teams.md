# <a name="access-a-vm-student-view-in-azure-lab-from-teams"></a>팀의 Azure 랩에서 VM(학생 보기)에 액세스

팀 내에서 랩을 만들면 사용자는 팀 소유자가 프로비저닝한 모든 VM을 보고 해당 VM에 액세스할 수 있습니다. 랩을 게시하고 VM을 만들면 사용자가 자동으로 랩에 등록되고 Azure Lab Services에 처음 로그인할 때 VM에 할당됩니다. 사용자는 **Azure Lab Services** 앱이 포함된 탭을 선택하여 자신에게 할당된 VM을 보고 해당 VM에 액세스할 수 있습니다.

랩을 아직 게시하지 않았거나 팀에 추가된 후 아직 동기화를 트리거하지 않은 경우 학생에게 비슷한 메시지가 표시됩니다.

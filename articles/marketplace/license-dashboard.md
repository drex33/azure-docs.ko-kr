# <a name="license-dashboard-in-commercial-marketplace-analytics"></a>상업용 Marketplace 분석의 라이선스 대시보드

이 문서에서는 파트너 센터에서 상업용 Marketplace 프로그램의 라이선스 대시보드에 대한 정보를 제공합니다. 라이선스 대시보드에는 다음과 같은 정보가 나와 있습니다.

- 라이선스를 구매한 고객 수
- 구매한 라이선스의 총 수
- 배포한 라이선스의 총 수
- 고객이 구매하고 배포한 라이선스 수
- 국가 및 지역에 걸친 라이선스 배포

## <a name="check-license-usage"></a>라이선스 사용 현황 확인

[!INCLUDE [Workspaces view note](./includes/preview-interface.md)]

#### <a name="workspaces-view"></a>[작업 영역 보기](#tab/workspaces-view)

파트너 센터에서 ISV 앱의 라이선스 사용 현황을 확인하려면 다음을 수행합니다.

1. [파트너 센터](https://partner.microsoft.com/dashboard/home)에 로그인합니다.
1. 홈 페이지에서 **Insights** 타일을 선택합니다.

    [![파트너 센터 홈 페이지의 Insights 타일을 보여 줍니다. ](./media/workspaces/partner-center-insights-tile.png) ](./media/workspaces/partner-center-insights-tile.png#lightbox)

1. 왼쪽 메뉴에서 **라이선스를** 선택합니다.

    [![파트너 센터 라이선스 대시보드의 스크린샷. ](./media/license-dashboard/license-dashboard-workspaces.png) ](./media/license-dashboard/license-dashboard-workspaces.png#lightbox)

#### <a name="current-view"></a>[현재 보기](#tab/current-view)

파트너 센터에서 ISV 앱의 라이선스 사용 현황을 확인하려면 다음을 수행합니다.

1. [파트너 센터](https://go.microsoft.com/fwlink/?linkid=2165507)에 로그인합니다.
1. 왼쪽 탐색 메뉴에서 **상업용 Marketplace** > **분석** > **라이선스** 를 선택합니다.

:::image type="content" source="./media/license-dashboard/license-dashboard.png" alt-text="파트너 센터의 라이선스 대시보드 스크린샷.":::

## <a name="elements-of-the-license-dashboard"></a>라이선스 대시보드의 요소

다음 섹션에서는 라이선스 대시보드를 사용하는 방법과 데이터를 읽는 방법을 설명합니다.

## <a name="month-range"></a>월 범위

페이지의 오른쪽 위 모서리에서 월 범위 선택 항목을 찾을 수 있습니다. 지난 6개월 또는 12개월을 기준으로 월 범위를 선택하거나 최대 기간이 12개월인 사용자 지정 월 범위를 선택하여 페이지에서 위젯 출력을 사용자 지정합니다. 기본 월 범위(계산 기간)는 6개월입니다.

#### <a name="workspaces-view"></a>[작업 영역 보기](#tab/workspaces-view)

#### <a name="current-view"></a>[현재 보기](#tab/current-view)

## <a name="customers-widget"></a>고객 위젯

_고객 위젯_ 은 현재 고객의 수를 보여 줍니다. 추세 차트는 월 단위 고객 수를 표시합니다.

:::image type="content" source="./media/license-dashboard/customers-widget.png" alt-text="파트너 센터의 라이선스 대시보드에 있는 고객 위젯의 스크린샷.":::

## <a name="license-widget"></a>라이선스 위젯

_라이선스 위젯_ 은 현재 프로비전되고 할당된 라이선스의 수를 보여 줍니다. 추세 차트는 프로비전되고 할당된 월 단위 라이선스의 수를 보여 줍니다. 최근 6개월 또는 12개월 동안의 데이터를 볼 수 있습니다. 페이지의 오른쪽 위에 있는 **사용자 지정** 링크를 선택하여 사용자 지정 날짜 범위를 선택할 수도 있습니다.

:::image type="content" source="./media/license-dashboard/license-widget.png" alt-text="파트너 센터의 라이선스 대시보드에 있는 라이선스 위젯의 스크린샷.":::

## <a name="analysis-widget"></a>분석 위젯

_분석 위젯_ 은 프로비전되고 할당된 라이선스의 수 및 백분율을 제품 및 플랜별로 보여 줍니다. 추세 차트는 프로비전되고 할당된 월 단위 라이선스의 수를 보여 줍니다. 이 위젯을 사용하여 고객 또는 제품을 기준으로 데이터를 필터링할 수 있습니다.

:::image type="content" source="./media/license-dashboard/analysis-widget.png" alt-text="파트너 센터의 라이선스 대시보드에 있는 분석 위젯의 스크린샷.":::

## <a name="license-distribution-widget"></a>라이선스 배포 위젯

_라이선스 배포_ 위젯은 다양한 국가 및 지역에 걸친 라이선스의 배포를 보여 줍니다. 색이 지정된 영역은 사용자 라이선스가 배포된 위치를 나타냅니다. 기본 보기로 되돌리려면 위젯에서 **확대/축소 다시 설정** 단추(홈 아이콘)를 선택합니다.

:::image type="content" source="./media/license-dashboard/license-distribution.png" alt-text="파트너 센터의 라이선스 대시보드에 있는 라이선스 배포 위젯의 스크린샷.":::

## <a name="data-terms-in-license-report-downloads"></a>라이선스 보고서의 데이터 용어 다운로드

위젯의 오른쪽 위 모서리에 있는 다운로드 아이콘을 사용하여 데이터를 다운로드할 수 있습니다.

| 특성 이름 | 정의 |
| ------------ | ------------- |
| 고객 국가 | 고객의 청구 국가 |
| 고객 국가 번호 | 고객의 청구 국가 번호 |
| 고객 이름 | 고객 이름 |
| 활성화 날짜 | 라이선스가 활성화된 날짜 |
| 제품 표시 이름 | AppSource에 표시되는 제품 제목 |
| Product ID | Product ID |
| 프로비전된 라이선스 | 고객 계정에 활성화된 라이선스 수 |
| 라이선스가 할당됨 | 고객이 해당 사용자에게 할당한 라이선스 수 |
| SKU 이름 | 제품에서 플랜의 이름 |
| 테넌트 ID | 테넌트의 고유 ID |
| 라이선스 상태 | 라이선스 상태 |
| 서비스 ID | 플랜을 라이선스 검사에 매핑하기 위해 패키지에서 사용되는 고유 식별자입니다. |

## <a name="fork-and-clone-the-repository"></a>리포지토리 포크 및 복제

포크 <https://github.com/microsoft/FluidHelloWorld> 단추를  클릭하여 FluidHelloWorld 리포지션의 고유한 포크를 만듭니다.

그런 다음, 다음 명령을 사용하여 로컬 컴퓨터에 포크를 복제합니다.

```shell
git clone -b main-azure https://github.com/<YOUR_GITHUB_ACCOUNT_NAME>/FluidHelloWorld.git
```

`<YOUR_GITHUB_ACCOUNT_NAME>`을 GitHub 사용자 이름으로 바꿔야 합니다.

다음으로, Visual Studio Code를 열고 **파일 > 폴더 열기** 로 이동하여 복제한 리포지토리를 편집기에서 엽니다.

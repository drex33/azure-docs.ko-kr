# <a name="manage-smart-groups-preview"></a>스마트 그룹 관리(미리 보기)

[스마트 그룹(미리 보기)은](./alerts-smartgroups-overview.md?toc=%2fazure%2fazure-monitor%2ftoc.json) 기계 학습 알고리즘을 사용하여 동시 발생 또는 유사성을 기준으로 경고를 그룹화하므로 사용자는 각 경고를 개별적으로 관리하는 대신 스마트 그룹을 관리할 수 있습니다. 이 문서에서는 Azure Monitor에서 스마트 그룹을 액세스하고 사용하는 방법을 안내합니다.

1. 경고 인스턴스에 대해 생성된 스마트 그룹을 보려면 다음 중 하나를 수행할 수 있습니다.

     1. **경고 요약** 페이지에서 **스마트 그룹을** 클릭합니다.    
    ![스크린샷은 스마트 그룹이 강조 표시된 경고 요약 페이지를 보여줍니다.](./media/alerts-managing-smart-groups/sg-alerts-summary.jpg)
    
     1. 모든 경고 페이지에서 스마트 그룹별 경고를 클릭합니다.   
     ![스크린샷은 스마트 그룹별 경고가 강조 표시된 모든 경고 페이지를 보여줍니다.](./media/alerts-managing-smart-groups/sg-all-alerts.jpg)

2. 이렇게 하면 경고 인스턴스에 대해 생성된 모든 스마트 그룹의 목록 보기로 이동됩니다. 여러 경고를 살펴보는 대신, 이제 스마트 그룹을 처리할 수 있습니다.   
![스크린샷은 모든 경고 페이지를 보여줍니다.](./media/alerts-managing-smart-groups/sg-list.jpg)

3. 스마트 그룹을 클릭하면 세부 정보 페이지가 열리며, 여기서 멤버 경고와 그룹화 이유를 확인할 수 있습니다. 이 집계를 사용하면 여러 경고를 살펴보는 대신 단일 스마트 그룹을 처리할 수 있습니다.   
![스크린샷은 세부 정보 페이지를 보여줍니다.](./media/alerts-managing-smart-groups/sg-details.jpg)

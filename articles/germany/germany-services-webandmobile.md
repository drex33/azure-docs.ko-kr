# <a name="azure-germany-web-and-mobile-services"></a>Azure 독일 웹 및 모바일 서비스

[!INCLUDE [closureinfo](../../includes/germany-closure-info.md)]

## <a name="app-service"></a>App Service
Azure App Service는 일반적으로 Azure 독일에서 사용할 수 있습니다.
### <a name="variations"></a>변형
Azure 독일에서 만든 Azure App Service 앱의 주소는 글로벌 Azure용으로 만든 주소와 다릅니다.

| 서비스 유형 | 글로벌 Azure | Azure 독일 |
| --- | --- | --- |
| App Service |*.azurewebsites.net |*.azurewebsites.de|

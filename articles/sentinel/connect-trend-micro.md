# <a name="connect-trend-micro-deep-security"></a>Trend Micro Deep Security 연결

Trend Micro Deep Security 커넥터를 사용하면 Deep Security 로그를 Azure Sentinel에 간편하게 연결하고, 대시보드를 보고, 사용자 지정 경고를 만들고, 조사를 개선할 수 있습니다. 이렇게 하면 조직의 네트워크와 시스템에 대한 더 많은 인사이트를 제공하고 보안 작업 기능을 개선할 수 있습니다.

[!INCLUDE [reference-to-feature-availability](includes/reference-to-feature-availability.md)]


## <a name="prerequisites"></a>필수 구성 요소

- Azure Sentinel 작업 영역에 대한 읽기 및 쓰기 권한.

## <a name="connect-azure-sentinel-to-trend-micro-deep-security"></a>Trend Micro Deep Security에 Azure Sentinel 연결

1. Azure Portal에서 **Azure Sentinel**  >  **데이터 커넥터** 로 이동한 다음 **Trend Micro Deep Security** 커넥터를 선택합니다.

2. **커넥터 페이지 열기** 를 선택합니다.

3. 지침 및 다음 단계를 따르세요.

### <a name="supported-regions"></a>지원되는 지역

#### <a name="azure-purview-management-side"></a>Azure Purview(관리 쪽)
Purview 액세스 정책 기능은 모든 Azure Purview 지역에서 사용할 수 [있습니다.](https://azure.microsoft.com/global-infrastructure/services/?products=purview&regions=all)


#### <a name="azure-storage-enforcement-side"></a>Azure Storage(적용 쪽)
Purview 액세스 정책은 다음 Azure Storage 지역에서만 적용할 수 있습니다.
-   프랑스 중부
-   캐나다 중부

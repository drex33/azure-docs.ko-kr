# <a name="azure-firewall-certifications"></a>Azure Firewall 인증

다음 인증은 글로벌 Azure 및 Azure Government에 대한 것입니다.

## <a name="icsa-labs-corporate-firewall-certification"></a>ICSA Labs 회사 방화벽 인증

![ICSA 인증](media/overview/icsa-cert-firewall-small.png)

ICSA Labs는 네트워크 연결 디바이스는 물론 보안 및 상태 IT 제품에 대한 타사 테스트 및 인증 분야의 선두 공급업체입니다. 세계 최고의 기술 공급업체 대부분의 제품 규정 준수, 안정성 및 성능을 측정합니다.

Azure Firewall은 ICSA Labs 회사 방화벽 인증을 획득한 최초의 클라우드 방화벽 서비스입니다. Azure Firewall 인증 보고서는 [ICSA Labs 인증 테스트 및 감사 보고서](https://aka.ms/ICSALabsCertification)를 참조하세요. 자세한 정보는 [ICSA Labs 방화벽 인증 프로그램](https://www.icsalabs.com/technology-program/firewalls) 페이지를 참조하세요.

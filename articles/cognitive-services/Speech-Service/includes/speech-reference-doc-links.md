## <a name="reference-docs"></a>참조 문서

Speech Service는 두 개의 SDK를 제공합니다. 첫 번째 SDK는 기본 [Speech SDK](../speech-sdk.md)이며 Speech Service와 상호 작용하는 데 필요한 대부분의 기능을 제공합니다. 두 번째 SDK는 [Speech Devices SDK](../speech-devices-sdk.md)를 적절하게 명명한 디바이스에만 적용됩니다. 두 SDK는 여러 언어로 제공됩니다.

### <a name="speech-sdk-reference-docs"></a>Speech SDK 참조 문서

다음 목록을 사용하여 적절한 Speech SDK 참조 문서를 찾습니다.

- <a href="https://aka.ms/csspeech/csharpref" target="_blank" rel="noopener">C# SDK </a>
- <a href="https://aka.ms/csspeech/cppref" target="_blank" rel="noopener">C++ SDK </a>
- <a href="https://aka.ms/csspeech/javaref" target="_blank" rel="noopener">Java SDK</a>
- <a href="https://aka.ms/csspeech/pythonref" target="_blank" rel="noopener">Python SDK</a>
- <a href="https://aka.ms/csspeech/javascriptref" target="_blank" rel="noopener">JavaScript SDK</a>
- <a href="https://aka.ms/csspeech/objectivecref" target="_blank" rel="noopener">Objective-C SDK </a>

> [!TIP]
> Speech Service SDK는 적극적으로 유지 관리되고 업데이트됩니다. 변경 내용, 업데이트 및 기능 추가를 추적하려면 [Speech SDK 릴리스 정보](../releasenotes.md)를 참조하세요.

### <a name="speech-devices-sdk-reference-docs"></a>Speech Devices SDK 참조 문서

[Speech Devices SDK](../speech-devices-sdk.md)는 특정 디바이스에 대한 확장 기능이 있는 Speech SDK의 상위 집합입니다. Speech Devices SDK를 다운로드하려면 먼저 [개발 키트를 선택](../get-speech-devices-sdk.md#choose-a-development-kit)해야 합니다.

# <a name="get-support-for-datadog-resource"></a>Datadog 리소스에 대한 지원 받기

이 문서에서는 Datadog 리소스 작업 시 지원에 문의하는 방법을 설명합니다. 지원에 문의하기 전에 [일반적인 오류 수정](troubleshoot.md)을 참조하세요.

## <a name="contact-support"></a>지원에 문의

Azure Datadog 통합에 대해 고객 지원팀에 문의하려면 왼쪽 창에서 **새 지원 요청** 을 선택합니다. Datadog 포털 링크를 선택합니다.

:::image type="content" source="media/get-support/support-request.png" alt-text="새 Datadog 지원 요청 만들기" border="true":::

## <a name="next-steps"></a>다음 단계

잠재적인 해결 방법은 [일반적인 오류 수정](troubleshoot.md)을 참조하세요.

기존 Datadog 리소스를 변경하는 방법에 대한 자세한 내용은 [Datadog 리소스 관리](manage.md)를 참조하세요.

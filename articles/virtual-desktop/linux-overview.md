# <a name="thin-client-support"></a>Thin 클라이언트 지원

[웹 클라이언트](connect-web.md) 또는 파트너가 제공하는 다음과 같은 지원되는 클라이언트를 사용하여 씬 클라이언트 디바이스에서 Azure Virtual Desktop 리소스에 액세스할 수 있습니다. 다른 플랫폼에서 지원되는 Azure Virtual Desktop 클라이언트를 사용하도록 설정하기 위해 많은 파트너와 협력하고 있습니다.

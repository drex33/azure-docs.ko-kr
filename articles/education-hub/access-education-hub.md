# <a name="accessing-the-azure-education-hub"></a>Azure Education Hub 액세스

Azure Education Hub에 액세스하려면 교육용 권한 부여이 승인되었음을 알리는 이메일 알림이 이미 수신되어 있어야 합니다. 이메일에는 승인된 크레딧 금액 및 [Education Hub](https://aka.ms/startedu)에 대한 링크가 포함되어 있습니다. 대부분의 후원 교육용 제품은 Azure 지원 개발자 계층을 무료로 제공합니다. 설치 프로세스에 문제가 있는 경우 [Educator Services Desk](mailto:azuredu@microsoft.com)에 문의하거나 로컬 Microsoft 담당자에게 문의하세요.

> [!IMPORTANT]
> 교육용 권한 부여를 사용하려면 Education Hub에서 클래스룸을 만들고 Azure 크레딧에 액세스할 수 있는 클래스룸 내의 구독을 사용해야 합니다.

## <a name="how-to-access-the-azure-education-hub"></a>Azure Education Hub에 액세스하는 방법

1. 브라우저에서 Chrome의 Incognito 또는 Microsoft Edge의 InPrivate와 같은 개인 검색 탭을 엽니다.

    :::image type="content" source="media/access-education-hub/browsing-in-private.png" alt-text="개인 브라우저 창을 엽니다." border="false":::

1. Microsoft 자격 증명을 사용하여 [Azure Portal](https://portal.azure.com)에 로그인합니다.

   > [!IMPORTANT]
   > 조직/회사 계정(예: 교육 기관의 @domain.edu)을 사용하여 로그인하고 있는지 확인합니다. 확인했으면, 먼저 창의 왼쪽에서 이 옵션을 선택합니다. 그러면 다른 로그인 화면으로 이동합니다.

    :::image type="content" source="media/access-education-hub/sign-in.png" alt-text="조직 로그인 대화 상자." border="false":::

1. 로그인하면 Azure Portal로 이동합니다. Education Hub를 찾으려면 **모든 서비스** 메뉴로 이동하여 **Education** 을 검색합니다.
   처음 로그인하면 시작 페이지가 표시됩니다.

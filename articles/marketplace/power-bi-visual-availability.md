# <a name="define-the-availability-of-a-power-bi-visuals-offer"></a>Power BI 시각적 개체 제품의 가용성 정의

이 페이지에서는 시장 및 출시 날짜를 포함 하 여 제품을 사용할 수 있는 위치 및 방법을 정의할 수 있습니다.

## <a name="markets"></a>시장

제품을 사용할 수 있는 지역/국가를 지정하려면 **지역/국가 편집** 을 선택합니다.

:::image type="content" source="media/power-bi-visual/markets.png" alt-text="시장 선택을 포함 하 여 파트너 센터의 가용성 페이지를 표시 합니다.":::

여기에서 선택한 항목은 새로운 합병에만 적용 됩니다. 사용자가 나중에 제거 하는 시장에서 앱을 이미 사용 하 고 있는 경우이를 계속 사용할 수 있지만 해당 시장의 새 고객이 제품을 얻을 수는 없습니다.

>[!IMPORTANT]
>이러한 요구 사항이 여기 또는 파트너 센터에 나열되지 않은 경우에도 현지 법적 요건을 충족할 책임은 게시자에게 있습니다. 모든 지역/국가를 선택하는 경우에도 지역 법률, 제한 사항 또는 기타 요인을 통해 특정 제품이 일부 국가 및 지역에 나열되지 않을 수 있습니다.

왼쪽 탐색 메뉴에서 다음 탭으로 진행하기 전에 **초안 저장** 을 선택하고 **기술 구성** 을 선택합니다.

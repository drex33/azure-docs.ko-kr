한국에서 디바이스를 반송하는 경우 다음 단계를 수행합니다.

1. 서비스 요청 번호(Data Box 로컬 웹 UI의 **배송 준비** 페이지에 *참조 번호* 로 표시됨)를 기록해 둡니다.
1. 디바이스의 전원이 꺼져 있고 케이블이 모두 제거되어 있는지 확인합니다. 디바이스와 함께 제공된 전원 코드를 감아서 뒤쪽 칸에 단단히 꽂습니다.
1. 배송에 사용된 원래 상자를 사용하여 디바이스를 포장합니다. 반품 운송장이 포함되어 있는지 확인합니다.
1. 픽업을 예약하려면 근무 시간(월~금, 오전 10시~오후 5시) 중에 *Quantium Solutions International* 핫라인 070-8231-1418로 전화합니다. *Microsoft Azure 픽업* 및 서비스 요청 번호를 알려주고 수거를 준비합니다.

   픽업을 예약하기 위해 [microsoft@rocketparcel.com](mailto:microsoft@rocketparcel.com)으로 이메일을 보낼 수도 있습니다. 다음 템플릿을 사용합니다.

   ```
   To: microsoft@rocketparcel.com
   Subject: Microsoft Azure Pickup | Service Request Number: <ReferenceNumber>
   Body:
   * Service Request Number (reference number): <ReferenceNumber>
   * Requested pickup date: yyyy/mm/dd;time:HH MM
   * <If the consignment note is missing, note that here along with the sender and receiver information. If the consignment note was included, please delete this section.>
   ```  

1. 픽업을 예약할 때 문제가 발생하거나 추가 요금을 지불해야 하는 경우 Azure Data Box Operations([adbops@microsoft.com](mailto:adbops@microsoft.com))에 이메일을 보내 지원을 요청합니다. **제목** 줄에 주문 이름과 발생한 문제를 입력합니다.

운송업체에서 디바이스를 픽업하고 검사하면 Azure Portal의 주문 상태가 **픽업됨** 으로 업데이트되고 추적 ID가 표시됩니다.

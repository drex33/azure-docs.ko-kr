# <a name="azure-cdn-subscription-offers-and-bandwidth-throttling"></a>Azure CDN 구독 제품 및 대역폭 제한

일부 Azure CDN 제품은 특정 구독 유형에서만 사용할 수 있습니다. 대역폭 제한은 구독 유형에 따라 적용될 수도 있습니다.

## <a name="free-and-trial-subscription"></a>무료 및 평가판 구독

Microsoft의 Azure CDN은 평가판 구독에 사용할 수 있는 유일한 CDN 제품입니다. 이 유형의 구독에는 대역폭 제한이 적용됩니다.  
 
## <a name="pay-as-you-go"></a>종량제

Akamai의 Azure CDN은 종량제 구독에 사용할 수 없습니다.   
 
Microsoft 및 Verizon의 Azure CDN에 대한 대역폭은 구독이 올바른 상태로 확인되고 충분한 지불 내역이 있을 때까지 제한됩니다. 구독 상태를 확인하고 제한을 제거하는 프로세스는 첫 번째 지불을 받은 후 자동으로 발생합니다.   
 
지불했지만 제한이 제거되지 않은 경우 [지원에 문의](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade)하여 제거를 요청할 수 있습니다. 
  
## <a name="enterprise-agreements"></a>기업계약

모든 Azure CDN은 기업계약 구독에 사용할 수 있습니다. 기업계약 구독에는 대역폭 제한이 없습니다.  
 
## <a name="other-offer-types"></a>기타 제품 유형
 
종량제와 동일한 기능이 다음 계약 유형에 적용됩니다.

* Visual Studio 
* MSDN
* 학생
* CSP

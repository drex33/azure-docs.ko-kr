# <a name="language-support-for-text-analytics-for-health"></a>상태 Text Analytics 언어 지원

> [!NOTE]
> * 컨테이너는 API 엔드포인트 및 SDK와는 다른 모델 버전을 사용합니다.
> * 새 모델 버전이 릴리스되면 언어가 추가됩니다. Text Analytics for Health의 현재 [모델 버전](how-to/call-api.md#specify-the-text-analytics-for-health-model)은 다음과 같습니다.
>    * API 및 SDK: `2021-05-15`
>    * 컨테이너: `2021-03-01`

이 문서를 사용하여 Text Analytics 상태 및 해당 Docker 컨테이너에서 지원하는 자연어를 알아봅니다.

# <a name="azure-data-factory-whitepapers"></a>Azure Data Factory 백서 

[!INCLUDE[appliesto-adf-xxx-md](includes/appliesto-adf-xxx-md.md)]

백서를 통해 Azure Data Factory를 보다 심층적으로 알아볼 수 있습니다. 이 문서에서는 Azure Data Factory에 사용할 수 있는 백서 목록을 제공합니다.

| **백서** | **설명** |
| --- | --- |
|[Azure Data Factory - 클라우드 내 데이터 통합](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-data-integration-in-the-cloud/Azure_Data_Factory_Data_Integration_in_the_Cloud.pdf) | 이 백서에서는 Azure Data Factory를 통해 최신 데이터 웨어하우스를 빌드하고, 고급 분석 기능을 통해 지능형 SaaS 애플리케이션을 구동하고, SQL Server Integrations Services 패키지를 Azure로 리프트하는 방법을 설명합니다.|
|[Azure Data Factory를 사용하여 온-프레미스 관계형 데이터 웨어하우스에서 Azure로 데이터 마이그레이션](https://azure.microsoft.com/resources/data-migration-from-on-premise-relational-data-warehouse-to-azure-data-lake-using-azure-data-factory/) | 이 백서에서는 Azure Data Factory를 사용하여 기존 온-프레미스 관계형 데이터 웨어하우스(예: Netezza, Oracle, Teradata, SQL 서버)에서 Azure(예: Blob Storage 또는 Azure Data Lake Storage)로 수십 개의 TB 데이터를 마이그레이션하는 복잡성을 다룹니다. Azure Data Factory에서 Azure로의 빅 데이터 수집 과정에 대한 복원력, 성능, 확장성, 관리 및 보안과 관련된 과제와 모범 사례를 설명합니다. |
|[Azure Data Factory를 사용한 SAP 데이터 통합](https://github.com/Azure/Azure-DataFactory/blob/master/whitepaper/SAP%20Data%20Integration%20using%20Azure%20Data%20Factory.pdf) | 이 백서에서는 대상 시나리오, SAP 커넥터 옵션 및 다양한 요구 사항에 대한 비교, Data Factory의 각 SAP 커넥터 소개를 비롯하여 SAP 데이터 통합에 대한 Azure Data Factory의 현재 지원을 요약합니다. | 
|[Azure Data Factory: 클라우드의 SSIS](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-ssis-in-the-cloud/Azure_Data_Factory_SSIS_in_the_Cloud.pdf)| 이 백서에서는 기존 SSIS 워크로드를 Azure Data Factory로 마이그레이션하고 일반적인 고려 사항과 문제를 해결하려는 이유를 설명합니다. 그런 다음, Azure-SSIS IR을 만드는 방법에 대한 기술적인 세부 정보를 안내하고 SSMS(SQL Server Management Studio)와 같은 익숙한 도구를 사용하여 Azure Data Factory를 통해 패키지를 업로드, 실행 및 모니터링하는 방법을 보여줍니다. |
|[Azure Data Factory – 매개 변수 전달](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-passing-parameters/Azure%20data%20Factory-Whitepaper-PassingParameters.pdf)| 이 문서에서는 매개 변수를 작업 간에 전달하는 방법뿐만 아니라 파이프라인과 작업 간에 전달하는 방법을 안내합니다. |
|[Azure Data Factory – DevOps](https://azure.microsoft.com/mediahandler/files/resourcefiles/whitepaper-adf-on-azuredevops/Azure%20data%20Factory-Whitepaper-DevOps.pdf)| 이 백서에서는 Azure Data Factory의 연속 통합 및 배포와 관련된 몇 가지 모범 사례에 대해 설명합니다. |

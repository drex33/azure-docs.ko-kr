Azure 구독당 Cognitive Services 리소스 수에는 다음 한도가 적용됩니다. 각 Cognitive Services에는 다른 한도가 있을 수도 있습니다. 자세한 내용은 [Azure Cognitive Services](../articles/cognitive-services/index.yml)를 참조하세요.

| 유형 | 제한 | 예제 |
|------|-------|---------|
| Cognitive Services 리소스 혼합 | 지역별 Cognitive Services 리소스 최대 총 200개 | 미국 서부의 Computer Vision 리소스 100개, 미국 서부의 Speech Service 리소스 50개, 미국 서부의 Text Analytics 리소스 50개 |
| Cognitive Services 리소스의 단일 형식 | 지역별 리소스 최대 100개 | 미국 서부 2의 Computer Vision 리소스 수 100개와 미국 동부의 Computer Vision 리소스 수 100개 |

**새 테이블** 대화 상자에서 다음과 같이 양식을 작성합니다.

1. 테이블 ID에 **WeatherData** 라는 이름을 입력합니다.  이는 테이블의 이름입니다.
1. 이 예제에서는 *테이블 처리량(자동 크기 조정)* 에서 **수동** 을 선택합니다.
1. 예상 RU/s에서 기본값인 400을 사용합니다.
1. **확인** 단추를 선택하여 테이블을 만듭니다.

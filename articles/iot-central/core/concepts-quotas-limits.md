# <a name="quotas-and-limits"></a>할당량 및 제한

IoT Central 애플리케이션에 적용되는 다양한 할당량 및 제한이 있습니다. IoT Central 애플리케이션은 내부적으로 IoT Hub 및 DPS(Device Provisioning Service)와 같은 여러 Azure 서비스를 사용하며, 이러한 서비스에는 할당량 및 제한도 있습니다. 관련된 경우 기본 서비스의 할당량 및 제한이 이 문서에서 호출됩니다.

## <a name="devices"></a>디바이스

| 항목 | 할당량 또는 제한 | 참고 |
| ---- | -------------- | ----- |
| 애플리케이션의 디바이스 수 | 1,000,000 | 지원 담당자에게 문의하여 애플리케이션에 대해 이 할당량을 늘리는 방법을 설명합니다. |

## <a name="telemetry"></a>원격 분석

| 항목 | 할당량 또는 제한 | 참고 |
| ---- | -------------- | ----- |
| 디바이스당 초당 원격 분석 메시지 수| 10 | 이 제한을 초과해야 하는 경우 지원 담당자에게 문의하여 애플리케이션에 대한 증가에 대해 논의합니다. |
| 디바이스-클라우드 메시지의 최대 크기 | 256KB | 이 값은 IoT Hub 서비스에서 설정됩니다. |
| 클라우드-디바이스 메시지의 최대 크기 | 64KB | 이 값은 IoT Hub 서비스에서 설정됩니다. |

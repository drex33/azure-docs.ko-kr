# <a name="open-a-support-ticket-for-an-importexport-job"></a>가져오기/내보내기 작업에 대한 지원 티켓 열기

Import/Export 서비스에 문제가 발생하는 경우 기술 지원을 위해 서비스 요청을 만들 수 있습니다.  

이 문서에서는 다음을 안내합니다.

* 지원 요청을 만드는 방법 
* 포털 내에서 지원 요청 주기를 관리하는 방법

## <a name="create-a-support-request"></a>지원 요청 만들기

지원 요청을 만들려면 다음 단계를 수행합니다.

1. 가져오기/내보내기 작업으로 이동합니다. **지원 + 문제 해결** 아래의 **새 지원 요청** 을 선택합니다.
     
    ![기본 사항](./media/storage-import-export-contact-microsoft-support/import-export-support-01.png)
   
1. **기본** 탭에서 필수 기본 정보를 입력합니다. Import/Export 주문에 대한 기본 정보가 이미 입력되어 있습니다.

     |옵션|Description|
     |------|-----------|
     |**요약** | 문제를 간략히 설명하세요.|
     |**문제 유형**|**기술적** 을 선택합니다.|
     |**구독**|Import/Export 주문에 대한 구독을 선택합니다. |
     |**서비스**|**내 서비스** 를 선택합니다.|
     | **서비스 종류**| **Azure Import/Export 서비스** 를 선택합니다.|
     |**리소스**|Import/Export 주문을 선택합니다.|
     |**문제 유형**|Import/Export 주문에 대한 문제 유형 중에서 문제 유형을 선택합니다.|
     |**문제 하위 유형**|문제 유형에 적합한 하위 유형을 선택합니다.|
 
     ![Import/Export 서비스 지원 요청에 대한 기본 사항 화면의 정보를 보여 주는 스크린샷. 솔루션 탭을 여는 단추가 강조 표시됩니다.](./media/storage-import-export-contact-microsoft-support/import-export-support-02.png)

   **다음: 솔루션>>** 을 선택하여 계속합니다.

1. **솔루션** 탭에 사용자가 설명한 문제에 대한 전문 솔루션이 표시됩니다. 문제에 대한 솔루션을 찾지 못하면 **다음: 세부 정보>>** 를 선택하여 지원 요청으로 진행합니다.

    ![Import/Export Service 지원 요청에 대한 솔루션 탭을 보여 주는 스크린샷. 솔루션 탭에 선택한 문제에 대한 전문 솔루션이 표시됩니다. 솔루션 탭을 여는 단추가 강조 표시됩니다.](./media/storage-import-export-contact-microsoft-support/import-export-support-03.png)

1. **세부 정보** 탭을 사용하여 지원 티켓에 대한 정보를 입력합니다.

    |Grouping(그룹화)|옵션|
    |--------|-------------------|
    |**문제 세부 정보**|<ul><li>**문제가 언제 시작되었나요?** 현지 시간을 사용하여 문제가 시작된 데이터와 시간을 입력합니다.</li><li>**세부 정보**: 사용자에게 발생한 문제를 설명합니다.</li><li>**파일 업로드**: 필요한 경우 하나 이상의 파일을 첨부할 수 있습니다. 폴더 아이콘을 선택합니다. 그런 다음, 업로드할 파일을 찾습니다.</li></ul>|
    |**진단 정보 공유**|<ul><li>**아니오** 를 선택합니다. Azure 지원은 필요한 경우 추가 정보를 요청합니다.</li></ul>|
    |**지원 방법**|<ul><li>**심각도** 를 **C - 최소 영향** 으로 둡니다. Azure 지원에서 사용자가 제공한 정보에 따라 심각도 수준을 조정합니다.</li><li>그렇지 않으면 이메일이나 전화를 통해 어떤 언어로 연락하고 싶은지 알려주세요.</li></ul>|
    |**연락처 정보**|<ul><li>필요한 경우 연락처 정보를 편집할 수 있습니다.</li></ul>|

    ![Import/Export 서비스 지원 요청에 대한 세부 정보 탭을 보여 주는 스크린샷. 검토 더하기 만들기 탭을 여는 단추가 강조 표시됩니다.](./media/storage-import-export-contact-microsoft-support/import-export-support-04.png)

    **다음: 검토 + 만들기 >>** 를 선택하여 계속 진행합니다.

1. **검토 + 만들기** 탭에서 지원 요청을 검토합니다. 필요한 내용을 변경합니다. 그런 다음 **만들기** 를 선택합니다. 

    ![새 Azure 지원 요청에 대한 검토 더하기 만들기 탭을 보여 주는 스크린샷. 만들기 단추가 강조 표시됩니다.](./media/storage-import-export-contact-microsoft-support/import-export-support-05.png)


## <a name="manage-a-support-request"></a>지원 요청 관리

지원 티켓을 만든 후에는 포털 내에서 티켓의 수명 주기를 관리할 수 있습니다.

#### <a name="to-manage-your-support-requests"></a>지원 요청을 관리하려면

1. 도움말 및 지원 페이지로 이동하려면 **찾아보기 > 도움말 + 지원** 으로 이동합니다.

    ![Azure Portal 홈페이지에서 “도움말 더하기 지원”을 선택하는 방법을 보여 주는 스크린샷.](./media/storage-import-export-contact-microsoft-support/manage-support-ticket-01.png)

2. **최근 지원 요청** 의 테이블 형식 목록이 **도움말 + 지원** 에 표시됩니다.

    ![열기 지원 요청이 강조 표시된 Azure “도움말 더하기 지원” 페이지를 보여 주는 스크린샷.](./media/storage-import-export-contact-microsoft-support/manage-support-ticket-02.png) 

3. 지원 요청을 클릭하여 선택합니다. 이 요청의 상태 및 세부 정보를 볼 수 있습니다. 이 요청의 후속 작업을 수행하려면 **+ 새 메시지** 를 선택합니다.

    ![Azure 지원 요청에 대해 선택한 새 메시지를 보여 주는 스크린샷.](./media/storage-import-export-contact-microsoft-support/manage-support-ticket-03.png)

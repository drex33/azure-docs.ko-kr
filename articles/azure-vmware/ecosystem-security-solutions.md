# <a name="security-solutions-for-azure-vmware-solution"></a>Azure VMware Solution에 대한 보안 솔루션

Azure VMware Solution 기본적인 부분은 보안입니다. 이를 통해 고객은 안전하고 신뢰할 수 있는 환경에서 VMware 기반 워크로드를 실행할 수 있습니다.

보안 파트너는 위협 방지 및 보안 검사와 같은 보안 에코시스템의 여러 측면을 다루는 VMware 기반 환경에서 업계 최고의 솔루션을 제공합니다. 고객은 온-프레미스 배포를 위해 VMware NSX-T와 통합된 이러한 솔루션을 많이 채택했습니다. 주요 원칙 중 하나로, Azure에서 실행되는 투자 및 VMware 솔루션을 계속 사용할 수 있게 하려고 합니다. 이러한 ISV(Independent Software Vendor)의 대부분은 Azure VMware Solution 통해 솔루션의 유효성을 검사했습니다.

이러한 솔루션에 대한 자세한 내용은 다음에서 확인할 수 있습니다.

- [Bitdefender](https://businessinsights.bitdefender.com/expanding-security-support-for-azure-vmware-solution)
- [Trend Micro Deep Security](https://www.trendmicro.com/en_us/business/products/hybrid-cloud/deep-security.html)
- [Check Point](https://www.checkpoint.com/cloudguard/cloud-network-security/iaas-public-cloud-security/)

# <a name="pay-for-your-azure-subscription-by-check-or-wire-transfer"></a>수표 또는 전신 송금을 통해 Azure 구독에 대한 요금 지불

이 문서는 MCA(Microsoft 고객 계약)를 사용하는 고객과 Azure 웹 사이트를 통해 Azure에 등록한 고객에게 적용됩니다(종량제 계정이라고도 하는 Microsoft Online Services 프로그램 계정의 경우). Microsoft 담당자를 통해 Azure에 등록한 경우 기본 결제 방법이 *이미 수표 또는 전신 송금으로* 설정되어 있습니다.

수표 또는 전신 송금을 통해 지불로 전환하는 경우 수표/전신 송금을 통해 청구서 날짜 30일 이내에 청구서를 지불한다는 의미입니다. 

결제 방법을 수표/전신 송금으로 변경하도록 요청하면 다음과 같은 두 가지 결과가 발생할 수 있습니다.

- 자동으로 승인되고 회사에 대한 정보를 묻는 메시지가 표시됩니다.
- 자동으로 승인되지는 않지만 Azure 지원에 요청을 제출할 수 있습니다.

> [!IMPORTANT]
> * 수표로 지불 또는 전신 송금은 회사를 대신하여 Azure를 사용하는 고객만 사용할 수 있습니다.
> * 수표 또는 전신 송금을 통해 지불로 전환하기 전에 미결제 요금을 모두 지불합니다.
> * 미결제 청구서는 기본 결제 방법으로 지불됩니다. 수표 또는 전신 송금으로 결제하려면 승인된 후 기본 결제 방법을 수표 또는 전신 송금으로 변경해야 합니다.
> * 현재 수표 또는 전신 송금을 통해 결제하는 것은 중국의 글로벌 Azure에서 지원되지 않습니다.
> * 수표 또는 전신 송금을 통해 지불로 전환하는 경우 신용 카드 또는 직불 카드로 결제로 다시 전환할 수 없습니다.

## <a name="request-to-pay-by-check-or-wire-transfer"></a>수표 또는 전신 송금으로 결제 요청

1. Azure Portal에 로그인합니다.
1. **구독으로** 이동한 다음, 수표 또는 전신 송금을 설정할 구독을 선택합니다.
1. 왼쪽 메뉴에서 **결제 방법** 을 선택합니다.
1. 결제 방법 페이지에서 **수표/전신 송금으로 결제를** 선택합니다.  
    :::image type="content" source="./media/pay-by-invoice/payment-methods.png" alt-text="결제 방법 페이지를 보여주는 스크린샷." lightbox="./media/pay-by-invoice/payment-methods.png" :::
1.  **수표/전신 송금으로 지불** 페이지에 신용 카드 또는 직불 카드를 사용하는 자동 결제 대신 수표/전신 송금을 사용하도록 요청할 수 있다는 메시지가 표시됩니다. **계속을** 선택하여 검사를 시작합니다.  
    
    :::image type="content" source="./media/pay-by-invoice/pay-check-wire-transfer-continue.png" alt-text="계속 옵션을 보여주는 스크린샷." :::
1. 승인 상태에 따라:  

    - 자동으로 승인되면 수표 또는 전신 송금으로 지불하도록 승인되었다는 메시지가 페이지에 표시됩니다. 회사 **이름을** 입력한 **다음, 저장을** 선택합니다.  
        :::image type="content" source="./media/pay-by-invoice/pay-check-wire-transfer.png" alt-text="수표/전신 송금 승인별 지불을 보여주는 스크린샷." lightbox="./media/pay-by-invoice/pay-check-wire-transfer.png" :::

    - 요청을 처리할 수 없거나 승인되지 않은 경우 다음 [수표 또는 전신 송금을 통해 지불을 설정하도록 요청 제출](#submit-a-request-to-set-up-pay-by-check-or-wire-transfer) 섹션의 단계를 수행해야 합니다.
1. 승인된 경우 결제 방법 페이지의 **기타 결제 방법** 에서 **수표 또는 전신 송금** 오른쪽에 있는 말줄임표(**...**) 기호를 선택한 **다음, 기본값으로 지정을** 선택합니다.  
    모두 수표 또는 전신 송금으로 지불하도록 설정됩니다.

## <a name="submit-a-request-to-set-up-pay-by-check-or-wire-transfer"></a>수표 또는 전신 송금을 통해 지불을 설정하는 요청 제출

자동으로 승인되지 않은 경우 수표 또는 전신 송금을 통해 결제를 승인하도록 Azure 지원에 요청을 제출할 수 있습니다. 요청이 승인되면 Azure Portal 수표 또는 전신 송금을 통해 지불로 전환할 수 있습니다.

1. Azure Portal에 로그인하여 지원 요청을 제출합니다. **도움말 + 지원** 을 검색하고 선택합니다.  
    :::image type="content" source="./media/pay-by-invoice/search-for-help-and-support.png" alt-text="도움말 및 지원을 검색하는 스크린샷." lightbox="./media/pay-by-invoice/search-for-help-and-support.png" :::
1. **새 지원 요청** 을 선택합니다.  
    :::image type="content" source="./media/pay-by-invoice/help-and-support.png" alt-text="새 지원 요청 링크의 스크린샷." lightbox="./media/pay-by-invoice/help-and-support.png" :::
1. **문제 유형** 으로 **청구** 를 선택합니다. *문제 유형* 은 지원 요청 범주입니다. 수표 또는 전신 송금으로 지불하려는 구독을 선택하고 지원 플랜을 선택한 후 **다음을** 선택합니다.
1. **결제** 를 **문제 유형** 으로 선택합니다. *문제 유형* 은 지원 요청 하위 범주입니다.
1. **청구서로 지불로 전환** 을 **문제 하위 유형** 으로 선택합니다.
1. **세부 정보** 상자에 다음 정보를 입력하고 **다음** 을 선택합니다.
    - 신규 또는 기존 고객:
    - 기존 고객인 경우 현재 결제 방법:
    - 주문 ID(청구서 옵션에 대한 요청):
    - 계정 관리자 Live ID(또는 Org ID)(회사 도메인이어야 함):
    - 상거래 계정 ID:
    - 회사 이름(VAT 또는 정부 웹 사이트에 따라 등록된 이름):
    - 회사 주소(VAT 또는 정부 웹 사이트에 따라 등록된 주소):
    - 회사 웹 사이트:
    - 국가:
    - 세금 ID/VAT ID:
    - 회사 설립(연도):
    - Microsoft와의 이전 비즈니스:
    - 담당자 이름:
    - 담당자 전화 번호:
    - 담당자 이메일:
    - 신용 카드 대신 수표 또는 전신 송금 결제 옵션을 원하는 이유에 대한 이유:
    - 코어 증가에 대해 다음과 같은 추가 정보를 제공합니다.
        - (이전 할당량) 기존 코어:
        - (새 할당량) 요청된 코어:
        - 특정 지역 및 일련의 구독:
    - **회사 이름** 과 **회사 주소** 는 Azure 계정에 대해 제공한 정보와 일치해야 합니다. 정보를 보거나 업데이트하려면 [Azure 계정 프로필 정보 변경](change-azure-account-profile.md)을 참조하세요.
    - Azure Portal에서 청구 연락처 정보를 추가해야 크레딧 한도를 승인받을 수 있습니다. 연락처 세부 정보는 회사의 지급 계정 또는 재무 부서와 관련되어 있어야 합니다.
1. 연락처 정보와 기본 연락 방법을 확인한 다음, **만들기** 를 선택합니다.

필요한 신용 수준으로 인해 신용 검사를 실행해야 할 경우 신용 검사 신청서를 보내드립니다. 회사의 감사된 재무 제표를 제공하도록 요청할 수 있습니다. 재무 정보를 제공하지 않거나 정보가 필요한 신용 한도를 지원하기에 충분하지 않은 경우 신용 조회 요청을 승인하기 위해 보증금 또는 신용의 대기 신용장을 요청할 수 있습니다.

## <a name="switch-to-pay-by-check-or-wire-transfer-after-approval"></a>승인 후 수표 또는 전신 송금으로 지불로 전환

Microsoft Online Services 프로그램(종량제) 계정이 있고 수표 또는 전신 송금으로 지불하도록 승인된 경우 Azure Portal 결제 방법을 전환할 수 있습니다.

Microsoft 고객 계약을 사용하면 청구 프로필을 수표/전신 송금으로 전환할 수 있습니다.

### <a name="switch-azure-subscription-to-checkwire-transfer"></a>Azure 구독을 수표/전신 송금으로 전환

아래 단계에 따라 수표 또는 전신 송금을 통해 지불하도록 Azure 구독을 전환합니다. *수표 또는 전신 송금을 통해 결제로 전환하면 신용 카드로 다시 전환할 수 없습니다.*

1. Azure Portal로 이동하여 계정 관리자 권한으로 로그인합니다. **Cost Management + 청구** 를 검색하고 선택합니다.  
    :::image type="content" source="./media/pay-by-invoice/search.png" alt-text="Azure Portal 내에서 Cost Management + Billing에 대한 검색을 보여주는 스크린샷." lightbox="./media/pay-by-invoice/search.png" :::
1. 수표 또는 전신 송금을 통해 지불로 전환하려는 구독을 선택합니다.
1. **결제 방법** 을 선택합니다.
결제 방법 페이지에서 **수표/전신 송금으로 결제를** 선택합니다.  
    :::image type="content" source="./media/pay-by-invoice/payment-methods.png" alt-text="수표/전신 송금으로 지불 옵션을 보여주는 스크린샷." lightbox="./media/pay-by-invoice/payment-methods.png" :::

### <a name="switch-billing-profile-to-checkwire-transfer"></a>청구 프로필을 수표/전신 송금으로 전환

청구 프로필을 수표/전신 송금으로 전환하려면 다음 단계를 수행합니다. 청구 프로필에 대한 기본 결제 방법을 변경하려면 Azure에 가입한 사람이어야 합니다.

1. Azure Portal로 이동하여 청구 정보를 확인합니다. **Cost Management + 청구** 를 검색하고 선택합니다.
1. 메뉴에서 **청구 프로필** 을 선택합니다.  
    :::image type="content" source="./media/pay-by-invoice/billing-profile.png" alt-text="청구 프로필 메뉴 항목을 보여주는 스크린샷." lightbox="./media/pay-by-invoice/billing-profile.png" :::
1. 청구 프로필을 선택합니다.
1. **청구 프로필** 메뉴에서 **결제 방법** 을 선택합니다.  
   :::image type="content" source="./media/pay-by-invoice/billing-profile-payment-methods.png" alt-text="결제 방법 메뉴 항목을 보여주는 스크린샷." lightbox="./media/pay-by-invoice/billing-profile-payment-methods.png" :::
1. *기타 결제 방법* 제목 아래에서 생략 부호(...)를 선택한 다음, **기본값으로 지정** 을 선택합니다.  
    :::image type="content" source="./media/pay-by-invoice/customer-led-switch-to-invoice.png" alt-text="확인/유선 전송 생략 부호 및 기본값 설정 옵션을 보여주는 스크린샷." lightbox="./media/pay-by-invoice/customer-led-switch-to-invoice.png" :::

제공한 정보가 불완전하거나 확인할 수 없는 경우 Microsoft에서 법적 설명서를 필요로 하는 경우가 있습니다. 다음과 같은 예가 있습니다.

* 계정 이름과 회사 이름 간의 이름 차이
* 이름 변경

프로젝트가 초기화되면 Azure에서 해당 프로젝의 실행을 시작합니다. 기본 **데이터 레이블** 지정 페이지에서 프로젝트를 선택 하 여 프로젝트에 대 한 세부 정보를 확인 합니다.

프로젝트를 일시 중지하거나 다시 시작하려면 오른쪽 위에서 **실행 중** 상태를 전환합니다. 프로젝트가 실행되는 경우에만 레이블을 데이터에 지정할 수 있습니다.

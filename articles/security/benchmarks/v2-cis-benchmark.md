# <a name="mapping-of-azure-security-benchmark-v2-and-cis-microsoft-azure-foundations-benchmark"></a>Azure 보안 벤치마크 v2 및 CIS Microsoft Azure 기초 벤치마크 매핑

CIS Microsoft Azure 기초 벤치마크는 Azure에 대해 보안 기준 구성을 설정할 수 있도록 CIS(Center for Internet Security)에서 제공하는 보안 지침입니다. 벤치마크의 범위는 Azure Cloud를 채택하는 동안 보안의 기초 수준을 설정하기 위한 것입니다. 이 벤치마크는 [Azure 보안 벤치마크 v2](overview.md)와 일치합니다.

당사는 [CIS Microsoft Azure 기초 벤치마크 최신 버전 v1.3.0](https://www.cisecurity.org/benchmark/azure/)과 [Azure 보안 벤치마크 v2](overview.md) 간의 매핑을 제공합니다. 이는 두 벤치마크에 대한 권장 사항이 어떻게 일치하는지 이해하는 데 도움을 줍니다. [여기](https://github.com/MicrosoftDocs/SecurityBenchmarks/blob/master/Azure%20Security%20Benchmark/2.0/asb_v2_to_cis_microsoft_azure_foundations_benchmark_v1.3.0.xlsx)에서 전체 매핑을 다운로드할 수 있습니다.

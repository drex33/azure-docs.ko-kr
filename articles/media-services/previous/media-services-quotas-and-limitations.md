이 문서에서는 Microsoft Azure Media Services와 관련된 할당량 및 제한 사항에 대해 설명합니다.

[!INCLUDE [azure-mediaservices-limits](../../../includes/azure-mediaservices-limits.md)]

## <a name="open-a-support-ticket-to-request-changes-to-the-default-quotas"></a>지원 티켓을 열어 기본 할당량에 대한 변경 요청
제공된 기본 할당량에 대한 변경을 요청하기 위해 지원 티켓을 열 수 있습니다. 원하는 할당량 변경, 사용 사례 시나리오 및 필요한 영역에 대한 세부 정보를 요청에 포함합니다.

### <a name="how-to-open-a-support-ticket"></a>지원 티켓을 여는 방법
Azure Portal에서 [도움말 + 지원](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/newsupportrequest)으로 이동합니다. Azure에 로그인하지 않은 경우 자격 증명을 입력하라는 메시지가 표시됩니다.

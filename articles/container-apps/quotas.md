# <a name="quotas-for-azure-container-apps-preview"></a>Azure 컨테이너 앱 미리 보기의 할당량

Azure Container Apps Preview에 대 한 구독 당 다음 할당량이 있습니다.

| 기능 | 수량 |
|---|---|
| 환경 | 2 |
| 환경 당 컨테이너 앱 | 20 |
| 컨테이너 앱 당 복제본 | 25 |
| 복제본 당 코어 수 | 2 |
| 환경 당 코어 수 | 50 |

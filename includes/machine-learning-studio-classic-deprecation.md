> 
> - [ML Studio(클래식)에서 Azure Machine Learning으로 기계 학습 프로젝트 이동에 대한 정보](../articles/machine-learning/migrate-overview.md)를 참조하세요.

> 
> ML Studio(클래식) 설명서는 사용 중지되며 나중에 업데이트되지 않을 수 있습니다.

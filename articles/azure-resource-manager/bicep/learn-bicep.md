# <a name="bicep-on-microsoft-learn"></a>Microsoft Learn의 Bicep

Bicep이 Azure 배포를 단순화하고 가속화하는 데 어떻게 도움이 되는지 확인할 준비가 되셨나요? Microsoft Learn에서 다양한 실습 과정을 확인합니다.

## <a name="get-started"></a>시작

Bicep을 접하는 경우 이 모듈을 Microsoft Learn 시작하는 것이 좋습니다.

Bicep을 통해 자동화되고 반복 가능한 방식으로 Azure 리소스를 구성하고 배포하는 방법을 더 쉽게 정의하는 방법을 알아봅니다. Bicep의 작동 방식을 직접 확인할 수 있도록 여러 Azure 리소스를 배포합니다. 개념을 연습하는 데 도움이 되는 Azure 리소스에 대한 무료 액세스를 제공합니다.

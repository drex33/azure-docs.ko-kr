# <a name="productivity-tools-on-the-data-science-virtual-machine"></a>Data Science Virtual Machine의 생산성 도구

DSVM에는 데이터 과학 및 프로그래밍 도구 외에도 동료들과 인사이트를 얻고 공유하는 데 도움이 되는 생산성 도구가 포함되어 있습니다. Microsoft 365는 가장 생산적이고 안전한 Office 환경으로, 팀이 언제 어디서나 원활하게 작업할 수 있게 해 줍니다. Power BI Desktop을 사용하여 데이터에서 인사이트로 그리고 또 동작으로 이동할 수 있습니다. 그리고 Microsoft Edge 브라우저는 최신의 빠르고 안전한 웹 브라우저입니다.

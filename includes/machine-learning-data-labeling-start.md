레이블 지정 프로젝트는 Azure Machine Learning에서 관리됩니다. **데이터 레이블 지정** 페이지를 사용하여 프로젝트를 관리합니다.

데이터가 이미 Azure Blob 스토리지에 있는 경우 레이블 지정 프로젝트를 만들기 전에 해당 스토리지를 데이터 저장소로 사용할 수 있도록 해야 합니다. 데이터 저장소 사용 예제는 [자습서: 첫 번째 이미지 분류 레이블 지정 프로젝트 만들기](../articles/machine-learning/tutorial-labeling.md)를 참조하세요.

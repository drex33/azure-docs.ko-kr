# <a name="what-is-the-azure-devtest-offer"></a>Azure Dev/Test 제공은 무엇인가요?

이 문서에서는 Visual Studio Subscription 혜택과 함께 제공되는 Azure Dev/Test 제공에 대해 설명합니다. 이 제공을 통해 다양한 Azure 서비스를 무료로 탐색하고 사용해 볼 수 있는 Azure 월간 크레딧을 받을 수 있습니다.  크레딧 금액은 구독 수준에 따라 다릅니다.  또한 사용자는 다양한 Azure 서비스에 대해 할인된 Azure Dev/Test 요금의 혜택을 받을 수 있으므로 비용 효율적인 개발 및 테스트가 가능합니다.

## <a name="who-can-benefit"></a>누가 혜택을 볼 수 있나요? 

Azure Dev/Test 제공을 통해 다음 Visual Studio 구독자는 Azure를 사용할 수 있습니다.
+ [개인은 Azure를 개발/테스트용 개인 샌드박스로 사용할 수 있습니다](quickstart-individual-credit.md). 
+ [기업은 팀 개발 및 공동 작업에 Azure를 사용할 수 있습니다](quickstart-create-enterprise-devtest-subscriptions.md).

# <a name="snpeff-genomic-variant-annotations-and-functional-effect-prediction-toolbox"></a>SnpEff: 게놈 변이 주석 및 기능성 효과 예측 도구 상자

[SnpEff](https://pcingola.github.io/SnpEff/) 유전적 변이 주석 및 기능성 효과 예측 도구 상자입니다. 유전자와 단백질의 유전적 변이(예: 아미노산 변화)의 효과를 예측하고 주석을 답니다.

데이터에 대한 자세한 내용은 [사용자 설명서](https://pcingola.github.io/SnpEff/se_introduction/)를 참조하세요.

[!INCLUDE [Open Dataset usage notice](../../includes/open-datasets-usage-note.md)]

## <a name="data-source"></a>데이터 원본

이 데이터 세트는 http://downloads.sourceforge.net/project/snpeff/databases/ 의 미러입니다.

## <a name="data-volumes-and-update-frequency"></a>데이터 볼륨 및 업데이트 빈도

이 데이터 세트는 약 2TB의 데이터를 포함하며 매월 업데이트됩니다.

## <a name="storage-location"></a>스토리지 위치

이 데이터 세트는 미국 서부 2 및 미국 중서부 Azure 지역에 저장됩니다. 선호도를 위해 미국 서부 2 또는 미국 중서부에 컴퓨팅 리소스를 할당하는 것이 좋습니다.

## <a name="data-access"></a>데이터 액세스

미국 서부 2: ‘ https://datasetsnpeff.blob.core.windows.net/dataset ’

미국 중서부: ‘ https://datasetsnpeff-secondary.blob.core.windows.net/dataset ’

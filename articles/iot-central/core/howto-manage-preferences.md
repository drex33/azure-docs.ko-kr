# <a name="manage-your-personal-application-preferences"></a>개인 애플리케이션 기본 설정 관리

IoT Central은 애플리케이션을 사용자 요구에 맞게 유연하게 사용자 지정할 수 있도록 합니다. 또한 IoT Central은 사용자별로 고유한 보기를 사용자 지정할 수 있도록 합니다. 이 문서에서는 사용자가 자신의 프로필에 적용할 수 있는 다양한 사용자 지정 옵션에 대해 설명합니다.

## <a name="change-language"></a>언어 변경

IoT Central은 여러 언어로 지원됩니다. 위쪽 탐색 모음의 설정 아이콘에서 **언어 선택** 을 사용하여 기본 설정 언어를 전환할 수 있습니다. 언어를 변경한 후에는 IoT Central이 선택한 언어를 기억해서 이를 모든 애플리케이션에 적용합니다. 대시보드 이미지와 같은 애플리케이션 내 사용자 지정은 지역화되지 않습니다.

## <a name="change-theme"></a>테마 변경

밝은 테마와 어두운 테마가 모두 지원됩니다. 밝은 테마는 기본값이지만, 상단 탐색 모음에서 설정 아이콘을 선택하여 테마를 변경할 수 있습니다.

![IoT Central 테마 선택](media/howto-manage-preferences/settings.png)

> [!NOTE]
> 관리자가 애플리케이션에 대해 사용자 지정 테마를 구성하면 밝은 테마와 어두운 테마 중에서 선택할 수 있는 옵션을 사용할 수 없습니다.

## <a name="change-default-organization"></a>기본 조직 변경

애플리케이션에서 조직을 사용하는 경우 조직을 선택해야 할 때마다 사용할 기본 조직을 선택할 수 있습니다. 예를 들어 기본 조직은 애플리케이션에 새 디바이스를 추가할 때 조직 필드를 미리 채웁 수 있습니다.

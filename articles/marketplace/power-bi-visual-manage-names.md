# <a name="manage-power-bi-visual-offer-names"></a>Power BI visual 제품 이름 관리

이 페이지에서는 제품의 추가 이름을 예약할 수 있습니다. 이러한 이름은 다른 언어로 사용 하거나 제품 이름을 변경 하려는 경우에 사용할 수 있습니다.

왼쪽 탐색 메뉴에서 **제안 관리** 를 선택 하 고 **제품 이름 관리** 를 선택 합니다.

:::image type="content" source="media/power-bi-visual/manage-product-names.png" alt-text="Power BI 시각적 제품에서 제품 이름을 관리 하는 탭을 표시 합니다.":::

## <a name="reserve-a-name"></a>이름 예약

1. **이름을** 입력 하 고 **사용 가능 여부 확인** 을 선택 하 여 해당 이름을 사용할 수 있는지 확인 합니다.
1. 이름을 사용할 수 있는 경우 **제품 이름 예약** 을 선택 하 여 이름 목록에 추가 합니다.
1. 예약 하려는 각 이름에 대해 반복 합니다.

이름을 삭제 하려면 **삭제** 를 선택 합니다.

제품 제출을 완료 하려면 이전 탭 (예: **제품 설정**)으로 돌아가서 페이지의 오른쪽 위 또는 아래에 있는 **검토 및 게시** 를 선택 합니다.

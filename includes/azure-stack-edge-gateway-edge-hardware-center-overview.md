디바이스를 조달하려면 Azure Edge 하드웨어 센터(미리 보기)로 이동하여 주문하세요. Azure Edge 하드웨어 센터 서비스를 사용하면 비즈니스 요구 사항에 따라 다양한 Azure Stack Edge SKU 중에서 선택할 수 있습니다. 디바이스 유형의 여러 단위를 주문하고, 여러 디바이스를 서로 다른 위치로 배송하고,, 향후 주문을 위해 주소를 저장하고, 주문의 상태를 추적할 수 있습니다. 

주문이 전달되면 디바이스를 구성하고 Azure Stack Edge 리소스를 만들어 디바이스를 관리할 수 있습니다.

## <a name="prerequisites"></a>사전 요구 사항

이 자습서를 진행하려면 사용자 지정 이미지를 만들려는 랩의 Azure Storage 계정에 업로드된 VHD(가상 하드 디스크) 파일이 필요합니다. 스토리지 계정에 VHD 파일을 업로드하려면 다음 문서 중 하나의 지침을 따릅니다.

- [AzCopy 명령줄 유틸리티 를 사용하여 VHD 파일을 업로드.](../articles/devtest-labs/devtest-lab-upload-vhd-using-azcopy.md)
- [Microsoft Azure Storage Explorer 사용하여 VHD 파일을 업로드.](../articles/devtest-labs/devtest-lab-upload-vhd-using-storage-explorer.md)
- [PowerShell 을 사용하여 VHD 파일을 업로드.](../articles/devtest-labs/devtest-lab-upload-vhd-using-powershell.md)

# <a name="what-is-conversational-language-understanding-preview"></a>대화형 Language Understanding(미리 보기)이란 무엇인가요?

대화형 Language Understanding은 최첨단 기술을 사용하여 대화형 언어 모델을 학습할 수 있는 클라우드 기반 서비스입니다. 

이 API는 개발 프로젝트를 위한 클라우드의 기계 학습 및 AI 알고리즘 모음인 [Azure Cognitive Services](../../index.yml)의 일부입니다. REST API 또는 클라이언트 라이브러리에서 이러한 기능을 사용할 수 있습니다.

## <a name="features"></a>기능

대화형 Language Understanding은 사용자 지정 컴퓨터 학습 인텔리전스를 사용자의 대화형 자연어 텍스트에 적용하여 전반적인 의미를 예측하고 관련성 있고 자세한 정보를 추출합니다.

* 고급 신경망을 이용한 고급 자연어 이해 기술.
* 강력하고 의미를 인식하는 분류 및 추출 모델.
* Language Studio를 사용하여 단순화된 모델 구축 경험.
* 한 언어로 교육하고 다른 언어로 테스트할 수 있는 기본적으로 다국어 모델입니다.
* 다른 대화형 Language Understanding 프로젝트, 사용자 지정 질문 답변 기술 자료 및 LUIS 애플리케이션을 비롯한 서비스를 연결할 수 있는 오케스트레이션 프로젝트 형식입니다.

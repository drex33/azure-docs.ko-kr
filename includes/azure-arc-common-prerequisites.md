## <a name="prerequisites"></a>사전 요구 사항

이 문서의 작업을 진행하려면 다음이 필요합니다.

- [클라이언트 도구 설치](../articles/azure-arc/data/install-client-tools.md)

   도구에는 Azure Data Studio 및 Azure CLI가 포함됩니다.

   Azure Data Studio의 경우 **Azure Arc** 및 **PostgreSQL** 용 확장을 데이터 서비스에 따라 설치합니다.
   
   Azure CLI의 경우 확장 `arcdata`를 설치합니다.

- [Azure Arc 데이터 컨트롤러 설치](../articles/azure-arc/data/create-data-controller.md)

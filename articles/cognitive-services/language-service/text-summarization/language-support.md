| 독일어                | `de`          | 2021-08-01                      |                    |
| 이탈리아어               | `it`          | 2021-08-01                      |                    |
| 일본어              | `ja`          | 2021-08-01                      |                    |
| 한국어                | `ko`          | 2021-08-01                      |                    |
| 스페인어               | `es`          | 2021-08-01                      |                    |
| 포르투갈어(브라질)   | `pt-BR`       | 2021-08-01                      |                    |

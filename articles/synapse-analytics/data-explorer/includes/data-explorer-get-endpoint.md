1. Synapse Studio의 왼쪽 창에서 **관리** > **Data Explorer 풀** 을 차례로 선택합니다.
1. 세부 정보를 보는 데 사용할 Data Explorer 풀을 선택합니다.

    :::image type="content" source="../media/ingest-data-pipeline/select-data-explorer-pool.png" alt-text="기존 풀 목록을 보여 주는 Data Explorer 풀 화면의 스크린샷":::

1. 쿼리 및 데이터 수집 엔드포인트를 기록해 둡니다. Data Explorer 풀에 대한 연결을 구성할 때 쿼리 엔드포인트를 클러스터로 사용합니다.
